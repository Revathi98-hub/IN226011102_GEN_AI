# Sentiment Analysis using NLP & ML 🚀

## 📌 Overview
This project builds an end-to-end sentiment analysis system using NLP preprocessing and machine learning models.

## ⚙️ Features
- Text preprocessing (cleaning, tokenization, stemming)
- Feature engineering (BoW, TF-IDF)
- ML models (Logistic Regression, Naive Bayes, Decision Tree)
- Model evaluation and comparison

## 🛠️ Tech Stack
- Python
- Scikit-learn
- NLTK

## 📊 Results
Logistic Regression achieved the best performance.
